PsychoReg

Psychologist's Front End to Children Behaviour Register

How does it work?

Psychologists logged in this web app can add children to their profiles, and view all the behaviours that a child's parents have registered through the mobile app.

When a child is added, the app generates a token associated to that particular child that the psychologist must provide to the child's parents.

Later, parents will add the child to their mobile apps using the token.


For testing and evaluation purposes, you can use these users that have the 'psychologist' role:

{username:'psycho2', password:'1234'}


Please, use these users fairly.
